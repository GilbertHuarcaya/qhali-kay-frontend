# QhaliKay

Web application to contact any hospital through chat, so you can generate queries or appointments.

## Technologies
Used:
- React
- Redux
- Axios
- Sass
- Auth0
- Chat Engine
- Eslint
- React-bootstrap
- React-simple-star-rating
- Mapbox-gl
- React-map-gl

## Installation

```bash
npm install

npm start
```
